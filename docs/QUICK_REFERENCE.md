# 🎯 Quick Reference: Indian Voice Models

## 🚀 **Instant Start (No Setup)**

Your app **already works** with Indian voices!

1. Open `http://localhost:3000`
2. Go to AI Interview → Settings
3. Select Indian voice (Aria, Kavya, Priya, Shreya, Arjun, Vikram, Rohit, Aditya)
4. Click "Test Voice"
5. **Done!** 🎉

---

## ⭐ **Want Premium Quality?**

### **3-Step Upgrade (5 minutes)**

1. **Get API Key** (FREE): [console.cloud.google.com](https://console.cloud.google.com)
   - Create project
   - Enable "Text-to-Speech API"
   - Create API Key

2. **Add to `.env` file:**
   ```
   REACT_APP_GOOGLE_TTS_API_KEY=your_key_here
   ```

3. **Restart:**
   ```bash
   npm run dev
   ```

**Result**: Authentic Indian accents! 🇮🇳
